# pokedexcli
A REPL CLI tool that allows interaction with PokeAPI to fetch Pokemon data through HTTP requests.

Built as part of the [boot.dev](https://www.boot.dev) backend developer roadmap.

## Features

### Chapter 1: REPL
- [x] Interactive REPL with command parsing
- [x] `help` and `exit` commands
- [x] `map` and `mapb` commands with pagination

### Chapter 2: Caching
- [x] PokeAPI integration with JSON deserialization
- [x] In-memory cache with concurrent-safe background cleanup
- [ ] Explore location areas
- [ ] Catch Pokemon
- [ ] Inspect caught Pokemon
